# 05. Process and Equipment Integration / 工艺与设备集成

## English
Computational lithography must interface closely with exposure tools, mask writers, and process control systems.  
Accurate OPC requires calibrated models linked to actual scanner optics and resist processes.

**Integration Aspects**
- Scanner lens and illumination settings  
- Photoresist process modeling  
- Overlay and metrology feedback loops  
- Digital twin and predictive maintenance links  

## 中文
计算光刻与曝光设备、掩膜版制备及工艺控制系统紧密相关。  
高精度 OPC 依赖于与实际扫描器光学系统和光刻胶工艺相匹配的模型。

**集成要点**
- 扫描器光学与照明参数建模  
- 光刻胶与蚀刻工艺建模  
- 叠层误差与计量反馈  
- 数字孪生与预测性维护的结合
