# 04. Machine Learning for OPC / 机器学习在 OPC 与 ILT 中的应用

## English
Recent research explores applying **Machine Learning (ML)** and **Deep Learning (DL)** to accelerate OPC and ILT.  
These models learn complex mappings between mask patterns and wafer contours, reducing computational cost.

**Representative Methods**
- CNN-based aerial image prediction  
- GANs for mask-to-wafer synthesis  
- Physics-Informed Neural Networks (PINNs) for optical modeling  
- Data augmentation and domain adaptation in lithography  

## 中文
近年来，研究者开始利用 **机器学习（ML）** 与 **深度学习（DL）** 加速 OPC 与 ILT 计算。  
模型通过学习掩膜与晶圆图形之间的映射关系，在保证精度的同时显著降低计算量。

**典型方法**
- 基于卷积神经网络（CNN）的成像预测  
- 基于生成对抗网络（GAN）的掩膜到晶圆生成  
- 基于物理约束神经网络（PINN）的光学建模  
- 数据增强与域自适应技术在光刻中的应用
