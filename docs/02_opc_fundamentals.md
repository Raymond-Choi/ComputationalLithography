# 02. OPC Fundamentals / 光学邻近效应修正基础

## English
Optical Proximity Correction (OPC) modifies mask shapes to compensate for optical and process distortions.  
There are two major categories:
1. **Rule-Based OPC:** Uses empirical corrections derived from pattern measurements.  
2. **Model-Based OPC:** Uses calibrated imaging models and numerical optimization.

**Core Techniques**
- Edge biasing and corner rounding correction  
- Calibration of lithography models (optical + resist + etch)  
- Evaluation metrics: CD error, EPE, process window  

## 中文
光学邻近效应修正（OPC）通过修改掩膜版图形来补偿光学与工艺造成的畸变。  
主要分为两类：  
1. **基于规则的 OPC**：依据经验公式与量测结果进行偏置修正；  
2. **基于模型的 OPC**：通过建立光学与抗蚀剂模型进行数值优化。

**核心技术**
- 线边偏置与拐角修正  
- 光刻模型（光学 + 光刻胶 + 蚀刻）标定  
- 关键尺寸误差（CD error）、边缘位置误差（EPE）、工艺窗口分析
