# 03. Inverse Lithography Technology (ILT) / 反向光刻技术

## English
Inverse Lithography formulates mask synthesis as an **inverse imaging problem** — finding a mask that produces the desired wafer contour.  
Mathematically, it minimizes the error between simulated and target patterns, often using gradient-based or level-set optimization.  

**Key Concepts**
- Cost function definition and regularization  
- Level-set and pixel-based optimization  
- Curvilinear mask shapes and manufacturability  

## 中文
反向光刻技术（ILT）将掩膜设计视为一个**成像逆问题**，通过迭代优化找到可生成目标晶圆图形的掩膜版。  
核心思想是在物理模型约束下最小化模拟图形与目标图形之间的误差。

**关键概念**
- 代价函数与正则化  
- 基于像素或轮廓的优化方法  
- 曲线化掩膜与可制造性限制
